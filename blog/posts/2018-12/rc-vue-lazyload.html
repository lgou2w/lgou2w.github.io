        安装

        在 VuePress 项目根目录运行命令.

        yarn add vue-lazyload // npm install vue-lazyload
        1

        配置

        打开项目内的 docs/.vuepress/enhanceApp.js 文件.

        import VueLazyload from 'vue-lazyload'

Vue.use(VueLazyload);
        1
        2
        3

        使用

        在需要用到懒加载图片的地方将 img 元素的 src 属性替换为 v-lazy 即可.

        <img src="test.jpg" alt="test" />
<!-- 替换为 -->
<img v-lazy="test.jpg" alt="test" />
